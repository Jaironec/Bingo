# 🎲 Bingo Multijugador - Edición Premium

Un juego de bingo multijugador en tiempo real con interfaz moderna, efectos visuales espectaculares y experiencia de usuario optimizada.

## ✨ Características Premium

### 🎮 **Funcionalidades de Juego**
- **🏠 20 tablas únicas**: Cada sala genera tablas completamente diferentes con algoritmo avanzado
- **👥 Hasta 20 jugadores**: Capacidad para grandes grupos con sincronización perfecta
- **🔒 Selección exclusiva**: Sistema inteligente que previene duplicados
- **🤖 Canto automático**: Velocidad configurable de 1-10 segundos
- **✋ Marcado interactivo**: Feedback visual inmediato en cada acción
- **🏆 5 patrones de victoria**:
  - 📏 **Línea**: Horizontal, vertical o diagonal
  - 🏁 **Tabla llena**: Victoria definitiva
  - 🔲 **Cuatro esquinas**: Patrón rápido
  - ❌ **Loco**: Exactamente 5 números
  - ⚡ **Machetaso**: Diagonal por el centro

- **🌟 Diseño glassmorphism**: Efectos de cristal y transparencias
- **🎭 Fondo animado**: Formas geométricas flotantes con gradientes
- **💎 Hero section**: Página de inicio impactante
- **📱 Responsive avanzado**: Optimización para todos los dispositivos
- **✨ Animaciones fluidas**: Transiciones suaves de 250ms
- **🎪 Efectos hover**: Feedback visual en todos los elementos
- **🌈 Gradientes modernos**: Sistema de colores profesional

### ⚙️ **Tecnología Avanzada**
- **🔄 WebSockets**: Comunicación en tiempo real sin latencia
- **🛡️ Verificación automática**: Sistema anti-trampa robusto

```bash

git clone https://github.com/tu-usuario/bingo-multijugador.git

# Instalar dependencias

# Abrir navegador
open http://localhost:3000
```

## 🎮 Guía de Juego

### 👑 **Para Anfitriones**
1. **🏗️ Crear sala**: Botón "Crear Nueva Sala" en la página principal
2. **⚙️ Configurar**: Selecciona patrones activos y velocidad del canto
3. **📤 Compartir**: Código de 4 dígitos generado automáticamente
4. **👥 Esperar**: Hasta que todos seleccionen sus tablas
5. **🚀 Iniciar**: Botón "Iniciar Juego" cuando estés listo

### 👤 **Para Jugadores**
1. **🚪 Unirse**: Ingresa el código de sala compartido
2. **📝 Nombre**: Identifícate con un nombre único
3. **🎯 Seleccionar**: Elige entre 20 tablas únicas disponibles
4. **🎲 Jugar**: Marca números cantados automáticamente
5. **🏆 Ganar**: Declara bingo al completar cualquier patrón

## 🎯 Patrones de Victoria

### 📏 **Línea** - Patrón Clásico
- **Horizontal**: 5 números en cualquier fila
- **Vertical**: 5 números en cualquier columna  
- **Diagonal**: 5 números en diagonal principal o secundaria

### 🔲 **Cuatro Esquinas** - Patrón Rápido
- Marca los 4 números de las esquinas de tu tabla
- Ideal para partidas cortas y emocionantes

### ❌ **Loco** - Patrón Libre
- Marca exactamente 5 números cualesquiera
- No importa la posición, solo la cantidad

### ⚡ **Machetaso** - Patrón Especial
- Diagonal que pase obligatoriamente por el centro
- Puede ser principal (↘️) o secundaria (↙️)

### 🏁 **Tabla Llena** - Victoria Total
- Marca todos los 25 números de tu tabla
- Termina definitivamente el juego
